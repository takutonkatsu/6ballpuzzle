# 6-Ball Puzzle Game - 要件定義書 (SPEC.md)

## 1. プロジェクト概要
* **ターゲット:** iOS / Android (Flutter開発)
* **画面向き:** 縦画面（ポートレート）専用
* **フェーズ1目標:** コアとなる操作感と連鎖システムを完備した「エンドレスモード」の完成。

## 2. コアゲームサイクル（エンドレスモード）
* 画面上部から3つ1組のボールが落下してくる。
* プレイヤーはボタンとタップ操作でボールを配置する。
* 同色が6個以上繋がると消滅し、スコアを獲得。
* 盤面上部（致死ライン）までボールが積み上がるとゲームオーバー。
* 時間経過とともにボールの基本落下速度が上昇する。

## 3. 画面レイアウト（上から順に配置）
1. **ヘッダー領域:** スコア、プレイ時間、設定ボタン。
2. **サブボード領域:** 次のボール（ネクスト、ダブルネクスト）を表示。
3. **メインボード領域:** プレイヤーが操作する盤面（六角形グリッドのマス目は非表示）。この領域全体が「ハードドロップ」用のタップエリアとなる。
4. **コントロール領域（最下部）:**
   * `[左回転]` `[左移動]` `[右移動]` `[右回転]` の4ボタンを配置。
   * 移動ボタンは長押しで連続移動とする。

## 4. プレイヤー操作とアクティブピース挙動
* **滑らかな移動:** 3つ1組のボール（アクティブピース）は、グリッドの列単位ではなく、ピクセル単位で滑らかに左右移動する。
* **落下予測（ゴースト）:** 「現在位置から真下にハードドロップした場合、最初に下の障害物（ボールや床）に激突する瞬間の3つ1組の姿」を半透明でリアルタイム表示する。判定は円形（Circle）で行う。
* **ハードドロップ:** メインボード領域をタップすると、ゴーストの位置まで一瞬で落下させる。

## 5. 消滅とワザ（特殊形状）のルール
* **基本条件:** 同色が6個以上隣接すると消去対象となる。
* **特殊形状（ワザ）の定義と優先度:**
  1. **ヘキサゴン（高）:** 中心1点の周囲を6個の同色ボールが囲むリング状。
  2. **ピラミッド（中）:** 底辺3、中間2、頂点1の正三角形。
  3. **ストレート（低）:** 任意の1軸に沿って6個が連続する直線。
* **ワザの効果:** ワザが成立した場合、繋がったボールだけでなく「盤面にある同色のボールを全消し」する。
