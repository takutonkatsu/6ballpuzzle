プライバシーポリシー

Takutonkatsu（以下、「開発者」といいます）は、スマートフォン向けアプリ「ヘキサゴン」（以下、「本アプリ」といいます）における、ユーザー情報の取扱いについて、以下のとおりプライバシーポリシー（以下、「本ポリシー」といいます）を定めます。

1. 収集する情報および利用目的

本アプリでは、ゲーム機能の提供、オンライン対戦、ランキング、広告配信、購入機能、品質向上のために、以下の情報を取得または保存する場合があります。

  • 端末内に保存される情報
    プレイヤー名、プレイヤーID、所持アイテム、コイン、EXP、音量や操作設定、戦績、ミッション進行状況などを、ゲームデータの保存および復元のために端末内へ保存します。
  • Firebase Authentication / Realtime Database (Google LLC)
    匿名認証ID、プレイヤー名、レート、ランキング情報、マッチング情報、対戦中の同期データ、スタンプ、通報・モデレーション関連情報、ギフトコード利用状況などを、オンライン対戦、ランキング、不正利用防止、ゲーム進行の同期のために利用します。
  • Google AdMob (Google LLC)
    バナー広告、インタースティシャル広告、リワード広告の配信および広告効果測定のため、広告ID等の識別子や端末情報が利用される場合があります。
  • アプリ内課金（Apple / Google）
    広告削除などの購入処理のため、各プラットフォームの決済サービスを利用します。開発者がクレジットカード番号等の決済情報を直接取得することはありません。

本アプリは、氏名、住所、電話番号など、ユーザー個人を直接特定する情報を開発者が意図して取得することはありません。

2. 個人情報の第三者提供

開発者は、法令に定められた場合を除き、ユーザーの同意なく個人情報を第三者へ提供することはありません。ただし、前項で述べたサードパーティサービスは、それぞれのプライバシーポリシーに基づき情報を取得・利用する場合があります。

3. データの削除について

端末内に保存されたゲームデータは、アプリの削除や端末設定からのデータ削除により削除できます。オンライン機能に関連して保存されたデータの削除をご希望の場合は、下記お問い合わせ窓口までご連絡ください。

4. プライバシーポリシーの変更

開発者は、必要に応じて本ポリシーを変更できるものとします。変更後のプライバシーポリシーは、本ページに掲載したときから効力を生じるものとします。
